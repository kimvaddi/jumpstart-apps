Welcome to Contoso Edge Store

Built by GitHub Actions | Deployed by Flux | Governed by Azure Arc

VERSION: 1.0

GitHub Actions builds and pushes the container to GHCR. Flux (running inside the K3s cluster as an Azure Arc managed extension) pulls the manifest from Git every 30 seconds and applies it to Kubernetes. Azure Arc exposes the GitOps configuration as an Azure resource for RBAC, Policy, and audit.
